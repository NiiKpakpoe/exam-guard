What this can and cannot do (please read)

A browser app cannot achieve OS-level lockdown, block a second phone, or run AI webcam proctoring like commercial tools. It can: enforce fullscreen, detect & log tab-switching/focus loss, block copy/paste/right-click and dev-tools shortcuts, randomize questions per student, enforce timers and a single attempt, optionally capture a webcam check-in photo, and produce a tamper-evident result file so scores cannot be forged. Treat it as a strong deterrent layered with good assessment design — always have a human review integrity flags before any decision.
